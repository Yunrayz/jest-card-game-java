Programme java permettant de jouer au jeu du Jest

La classe principale est la classe Main

Création d'extensions

Si on souhaite ajouter une extension, il faut :
1. Créer un package dans modele/extensions/
2. Créer la classe de notre nouvelle extension et implémenter l'interface Extension, puis créer nos nouvelles cartes, tel que fait avec l'extension ajoutant les cartes 6
Les valeurs pouvant être utilisées comme couleur et valeur de cartes sont définies dans les énumérations "CarteCouleur" et "CarteValeur", situées dans modele/cartes.
3. Une fois ceci fait, il faut ajouter l'extension dans la liste des extensions disponibles, dans la classe "Main" :
private static final Extension[] EXTENSIONS = {new ExtensionSix()};

Dépendances

Si vous souhaitez recompiler ce projet, il est nécessaire d'ajouter les librairies miglayout-core et miglayout-swing!
